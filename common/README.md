# common

Código compartido entre proyectos.

## Contenido esperado

- `config.py`: settings con `pydantic-settings`.
- `logging.py`: logging estructurado + request_id.
- `tracing.py`: hooks de tracing (LangSmith/Langfuse) en stub.
- `types.py`: modelos y dataclasses comunes.
- `eval/`: métricas y harness de evaluación reusable.
